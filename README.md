# Movie App(telemovie)

휴식 시간을 통하여 영화를 보고자 마음을 먹었으나 매번 어떤 영화를 보는 것이 좋을지 고민만 하게되는 사용자에게 무해한 이미지의 캐릭터와 한 눈에 잘 보이는 UI를 통하여 편안한 마음으로 영화 선택에 있어 조금 더 가볍고 빠르게 선택의 고민을 끝낼 수 있도록 돕고자 하는 목적으로 본 프로젝트를 진행하였습니다.

## 설치 항목

- [x] react-router-dom
- [x] styled-components
- [x] styled-reset
- [x] swiper
- [x] react-hook-form
- [x] react-helmet
